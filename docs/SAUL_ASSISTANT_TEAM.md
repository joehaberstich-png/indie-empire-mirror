## HOW IT WORKS

### Daily Cadence

```
07:00 — Maya sends daily briefing (top 3 decisions needed)
07:30 — James sends revenue dashboard (yesterday's sales, 7-day trend)
08:00 — Priya sends content pipeline status (blog posts due, episode progress)
08:30 — Diana sends compliance report (flags, violations, ban rates)
09:00 — Marcus sends infrastructure report (account health, backup integrity)
09:30 — Alex sends competitive memo (new entrants, price changes, opportunities)
10:00 — I review, decide, delegate back
```

### Weekly Cadence

```
Monday 09:00 — Strategy sync with Maya, Alex (30 min)
Wednesday 10:00 — Ops review with James, Marcus (30 min)
Friday 11:00 — Content review with Priya, Diana (30 min)
Friday 14:00 — CGO report to Megan via Maya's coordination
```

### Escalation Rules

Each assistant handles everything in their scope. Only escalate to me when:
1. Revenue drops >20% week-over-week
2. Platform ban rate spikes >2%
3. Backup integrity fails (red alert)
4. Competitor launches significant threat
5. Cross-department dispute can't resolve
