## Video Production Format

| Element | Specification |
|---------|---------------|
| Duration | 60-90 seconds (platform-optimized) |
| Format | 16:9 landscape + 9:16 vertical (both cuts) |
| Resolution | 4K (3840×2160) |
| Frame Rate | 24fps (cinematic) |
| Color Grade | Hollywood — teal/orange with cinematic LUT |
| Audio | Dolby 5.1 mix + stereo downmix |
| Voice | ElevenLabs — most human-like models (Rachel, Adam, Bella, Josh, Dani) |
| Intro | 3-second QBA TV bumper animation |
| Outro | 5-second QBA TV logo + "Subscribe at ContainerHomes.com" |

## A/B Testing Protocol

| Test Element | Variants | Sample Size | Winner Metric |
|-------------|----------|-------------|---------------|
| Voice actor | 3 per script | 500 views each | Watch time + click-through |
| Thumbnail | 4 per episode | 1,000 impressions | Click-through rate |
| Title | 3 per episode | 1,000 impressions | CTR + view duration |
| Intro length | 3s vs 5s vs 7s | 300 views | Retention at 0-10s |
| Call to action | End vs middle vs both | 500 views | Click-through rate |
| Music style | Cinematic vs upbeat vs ambient | 300 views | Watch time 0-60s |

**Detection Avoidance**:
- 6 voice variants per script — never reuse same voice+pace combination
- Random upload times (no batch scheduling)
- Thumbnail color rotation across a palette of 12 schemes
- Description templates rewritten 30% each post
- No duplicate metadata across episodes
