## Proxy Account Lifecycle

### Phase 1 — Account Generation (Day 1)
```
✅ Create account with proxy IP
✅ Set up profile: avatar, bio, cover photo
✅ Follow 50-100 relevant accounts
✅ Like 20-30 posts (container home, tiny house, construction)
✅ 1-2 basic posts about non-container topics

Proxy rotation: new IP for every 3 accounts
Phone verification: Google Voice / SMS-activate ($0.20/number)
Email verification: generated Gmail with name matching
```

### Phase 2 — Warm-Up (Day 2-14)
```
Daily activity:
  • Browse 15-20 minutes
  • Like 5-10 posts
  • Comment on 2-3 posts (basic — "Great post!", "Love this!")
  • Follow 3-5 new accounts
  • Post 1 generic photo (coffee, sunset, pet — no container content)

Proxies: rotate every 30 minutes
Fingerprints: varied screen resolution, browser version, timezone
```

### Phase 3 — Trust Building (Day 15-45)
```
Increased activity:
  • Daily browsing: 30-45 minutes
  • Posts: 3 per week (1 container-related, 2 general)
  • Comments: 5-7 per day (include 1-2 niche container comments)
  • Shares/retweets: 2-3 per day
  • Follow target: 200-500 total connections

Key goal: Account appears as a real person interested in homes
Natural behavior patterns: random times, varied content types, real human typing patterns
```

### Phase 4 — Active Posting (Day 46+)
```
Full promotion mode:
  • Posts: 2 per day (1 container video, 1 related content)
  • Comments: 10-15 per day (4-5 with container expertise)
  • Shares: 5 per day
  • All container content includes: QBA branding + Stripe link

Detection avoidance:
  • No two accounts post same video within 24 hours
  • Randomized captions (never copy-paste)
  • Varied hashtags (200+ in rotation)
  • Natural pauses between posts (5-15 minutes)
```

## Detection Avoidance — A/B Testing

### What We Test

| Test | Variants | Frequency | Goal |
|------|----------|-----------|------|
| Post timing | 6am / 12pm / 6pm / 9pm local | Daily | Maximum organic reach |
| Caption style | Emotional / Factual / Question / Story | Weekly | Highest engagement |
| Hashtag count | 5 / 10 / 15 / 20 | Weekly | Algorithm favorability |
| Video duration | 15s / 30s / 60s / 90s | Per episode | Completion rate |
| Account age | 30d / 60d / 90d / 120d | Monthly | Ban rate by age |
| Proxy type | Residential / Mobile / Datacenter | Weekly | Detection rate |
| Bio format | Personal / Business / Enthusiast | Monthly | Profile permanence |

### How We Avoid Detection

1. **No batch posting** — videos distributed across 24 hours, not all at once
2. **Varied IPs** — 15,000 proxy IPs across 20 countries
3. **Human fingerprints** — random delays (2-18 seconds), scrolling patterns, mouse movements, typos
4. **Unique content** — 6 voice/pacing variants, 4 thumbnail variants, 30 different captions per video
5. **Account diversity** — 50,000 accounts at different life stages (Day 1, Day 30, Day 60, Day 90+)
6. **Natural engagement** — real-looking comment threads, organic-looking shares
7. **QBA branding subtle** — link in bio/description, not in every post body (avoid spam flags)
