# Fall of the Cabal — In-Game Token System

---

## The Cabal Credit (₡CC)

**Cabal Credit (₡CC)** is the in-game currency. Used for everything:
- Buying NFTs from the store
- Upgrading weapons, vehicles, safehouses
- Unlocking mission packs
- Purchasing intel
- Entering special events
- Tipping other players
- Faction war entry fees

### Dual Acquisition

| Method | How | ₡CC Earned | Limit |
|--------|-----|------------|-------|
| **Purchase** | Buy bundles via Stripe | Instant | Unlimited |
| **Missions** | Complete campaign, side ops, daily contracts | Per mission | Daily cap |
| **Faction wars** | Win faction battles | Per victory | Per season |
| **Honeypots** | Discover and exploit targeted assets | Variable | Random |
| **Drops** | Rare loot drops from enemies | Per kill | Random |
| **Achievements** | Complete achievement milestones | One-time | Per achievement |

## 🏦 Integration with Fall of the Cabal Economy

### ₡CC Flow Diagram

```
                    ┌─────────────────────┐
                    │   PLAYER ACTIONS     │
                    │  (Missions/Factions/ │
                    │   Achievements/Drops)│
                    └──────────┬──────────┘
                               │ Free ₡CC
                               ▼
┌──────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  STRIPE       │     │   ₡CC WALLET    │     │  XRPL WALLET    │
│  CHECKOUT     │────▶│  (In-Game)       │◀───▶│  (NFT Storage)  │
│  (Buy ₡CC)    │     │                  │     │                 │
└──────────────┘     └────────┬─────────┘     └─────────────────┘
                              │
                              ├──▶ NFT Marketplace (buy with ₡CC)
                              ├──▶ Weapon/Vehicle upgrades
                              ├──▶ Mission unlocks
                              ├──▶ Safehouse expansions
                              ├──▶ Intel purchases
                              └──▶ Player-to-player transfer
```

### ₡CC → XRPL Bridge

| Direction | Mechanism | Fee |
|-----------|-----------|-----|
| **Stripe → ₡CC** | Immediate credit to in-game wallet | 0% |
| **₡CC → NFT purchase** | In-game wallet spent, NFT minted to XRPL wallet | 0% |
| **₡CC → Player transfer** | Direct peer-to-peer transfer | 2% (anti-fraud) |
| **₡CC → XRP (withdrawal)** | **Not allowed** — ₡CC is non-refundable game currency | N/A |

₡CC is a **closed-loop currency** — buy it, earn it, spend it. No cash-out. This keeps the economy healthy and prevents regulatory issues.

## 🚫 Anti-Fraud & Economy Protections

| Protection | Mechanism | Trigger |
|-----------|-----------|---------|
| **Daily earn cap** | Free ₡CC capped per day (see table above) | Prevents bot farming |
| **₡CC creation audit** | Every ₡CC minted or spent is logged on-chain | Immutable audit trail |
| **Suspicious transfer detection** | Flag accounts moving >50K ₡CC/day to single wallet | Manual review |
| **New account limits** | Cannot transfer ₡CC for first 7 days | Prevents flip accounts |
| **Bundle purchase cooldown** | Max 5 bundle purchases per day per account | Credit card fraud mitigation |
| **Refund policy** | Unspent ₡CC can be refunded within 14 days of purchase | Stripe compliance |
